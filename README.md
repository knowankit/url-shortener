## Requirements

1. [Node.js](https://nodejs.org/)
2. [npm](https://www.npmjs.com/)

## Steps to run this on your local

1. Clone this repo using `git clone https://github.com/knowankit/url-shortner.git`
2. Run `npm install`
3. Run `npm run dev`
